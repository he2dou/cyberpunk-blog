# React 19: Server Components 实战

## 为什么需要 RSC?

传统的 SSR (Server Side Rendering) 虽然解决了首屏渲染问题，但在 Hydration 阶段仍然需要下载和执行大量 JS。

**RSC (React Server Components)** 允许我们在服务器端运行 React 组件，直接发送 HTML 给客户端，且**不需要**在客户端 Hydrate 这些组件。
